## 📌 Project Overview

This project was developed during my internship to help restaurants make data-driven decisions by predicting profitability and optimizing operational strategies.

The solution leverages **Machine Learning**, **Business Analytics**, and an **Interactive Streamlit Dashboard** to provide actionable insights for restaurant managers.

### 🎯 Key Goals

✅ Predict Restaurant Profitability

✅ Analyze Multi-Channel Performance

✅ Perform What-If Scenario Analysis

✅ Visualize Channel Mix Distribution

✅ Generate Business Recommendations

---

## 🏢 Business Problem

Restaurants operate through multiple sales channels:

🏪 In-Store Orders

🛵 Uber Eats

🚚 DoorDash

📦 Self Delivery

Managing these channels efficiently is essential for maximizing profitability.

This project predicts restaurant profit and helps evaluate the impact of operational changes before implementing them.
